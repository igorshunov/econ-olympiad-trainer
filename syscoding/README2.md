### Задача 1

Дан массив целых чисел. Нужно найти максимальную сумму трёх неповторяющихся элементов массива.

**Примечания:**
- размер массива не меньше 3;
- элементы могут быть нулевыми или отрицательными;
- в массиве могут встречаться дубликаты, при суммировании они не учитываются.

**Примеры:**
- `maxTriSum({3,2,6,8,2,3}) -> 17`
- `maxTriSum({2,1,8,0,6,4,8,6,2,4}) -> 18`
- `maxTriSum({-7,12,-7,29,-5,0,-7,0,0,29}) -> 41`

### Задача 2

Для заданной строки нужно вернуть количество одиночных букв.

Буква считается одиночной, если:
- встречается во всей строке ровно один раз;
- у неё нет соседей по алфавиту в строке.

**Правила:**
- не учитывать регистр;
- не учитывать небуквенные символы;
- работать только с английскими буквами `a-z`.

**Примеры:**
- `"ad" -> 2`
- `"abc" -> 0`
- `"Hello, World!" -> 3`
- `"A-dA" -> 1`
- `"zz" -> 0`

### Задача 3

Прыгающее число — это число, в котором все соседние цифры отличаются на 1.

Нужно вернуть:
- `"Jumping!!"` — если число прыгающее;
- `"Not!!"` — если нет.

**Примечания:**
- передаваемое число всегда положительное;
- все однозначные числа считаются прыгающими;
- разница между цифрами `9` и `0` не считается за 1.

**Примеры:**
- `9 -> Jumping!!`
- `79 -> Not!!`
- `23 -> Jumping!!`
- `556847 -> Not!!`
- `4343456 -> Jumping!!`
- `89098 -> Not!!`
- `32 -> Jumping!!`

### Задача 4

Реализовать функцию `compress_images()`, которая:
- принимает на вход директорию;
- находит внутри неё картинки;
- уменьшает свойство `size` в их метаданных в 2 раза;
- возвращает обновлённую директорию со сжатыми картинками и всеми остальными данными.

Картинками считаются все файлы, заканчивающиеся на `.jpg`.

**Пример структуры:**
- `my documents`
  - `avatar.jpg` → `size: 100`
  - `photo.jpg` → `size: 150`

**Ожидаемый результат:**
- `avatar.jpg` → `size: 50`
- `photo.jpg` → `size: 75`
